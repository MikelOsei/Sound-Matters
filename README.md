# Sound Matters
Sound Matters is an interactive artwork that combines sound as input, with 3D visuals and ambient lighting to create an atmospheric scene that viewers can physically affect with their voices. This piece can repeat itself, but never in the same way, as there is a degree of randomness in the agentsâ€™ behaviour and state transitions.

Please see my artist statement for more details.

## Controls: 
* Interact with sound, allow mic permissions.
* Press H to show/hide the settings GUI
* Press R to reset the scene.
* Double click to show/hide the mouse pointer.
* Press SPACE to trigger the burst animation without using audio.
* Please adjust (lower) numAgents with the settings GUI if performance starts to slow.
